

## Features
- [x] 支持本地、网络音乐在线播放
- [x] 支持在线音乐榜单、电台、MV
- [x] 支持音乐在线播放、下载
- [x] 支持MV播放、下载
- [x] 支持本地及在线歌词
- [x] 支持在线搜索
- [x] 支持歌词、音乐缓存
- [x] 支持自建歌单、歌曲管理、收藏、排序、分类
- [x] 支持换肤、睡眠定时、模式切换、更多设置等
